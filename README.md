# Discensys Sessions

This repository contains materials from Discensys weekend sessions.

Discensys is a community-driven open learning collective focused on making high-quality learning and skill development accessible to everyone.

## What you'll find here

- Slides and resources from each session
- Topics selected through community voting
- Structured learning materials from always-free weekend sessions

## Repository Structure

Each session is organized by date and topic:

Year / Month / Session

Example:
2026 / 04-april / 2026-04-12-attention-mechanism

## Contributing

We welcome contributions from the community. If you'd like to contribute to sessions or materials, feel free to reach out.
